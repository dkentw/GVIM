#Install

1. Install vundle plugin for your VIM
2. Download the *vimrc* into the ``.vim`` folder
3. Open VIM and type the command ``BundllInstall``

It will install all needed plugin in vimrc.

Enjoy!

#Usage
**File Explorer**

``Ctrl+F9``

**Class Browser**

``Ctrl+F10``
